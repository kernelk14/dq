# DQ
A deque-based programming language
Inspired in [Porth](https://gitlab.com/tsoding/porth) and the [deque.py](https://github.com/tsoding/deque.py) project.

# Quick Start
```console
$ ./main.py hello.dq
```

# Language Basics
Print numbers:
`5 write`

Add numbers:
`5 5 + write`

Subtract numbers:
`6 5 - write`
